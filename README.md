# Student-Assessment-Analysis-Lab- (June 2025)
This project analyzes student demographic data and assessment information to gain insights into:
Student grade levels and age distribution
Types and weights of assessments
Relationships between student age bands and grade levels
The analysis includes data integration, cleaning, exploratory data analysis (EDA), and statistical testing, using Python and common data science libraries.

Key Features
1. Data Integration
Loads and merges student and assessment datasets
Standardizes column names
Formats student names

2. Data Cleaning & Preprocessing
Handles missing values
Converts date fields
Calculates student age and age bands
Validates emergency contact information

3. Exploratory Data Analysis (EDA)
Grade level distribution
Age band distribution
Assessment type distribution
Assessment weight histogram
Correlation heatmap

4. Statistical Analysis
Descriptive statistics
Chi-square test to analyze the relationship between Grade Level and Age Band

Technologies: Machine Learning, Python, Pandas, NumPy, Scikit-learn, Matplotlib, Jupyter Notebook
